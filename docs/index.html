📖 关于项目

Scrape API 由 PANGOLIN 精心打造，是一款专为跨境电商卖家、数据服务商和工具开发者设计的强大、稳定、高效的数据采集API。 我们的电商数据接口通过智能识别算法，能动态兼容 Amazon、Walmart 等各类电商平台页面结构的细微变化，自动识别并提取相关的产品数据，如商品标题、折扣信息、实时价格、库存状况和详细描述等。

开发者无需再为目标页面DOM结构的频繁变更而烦恼，Pangolin Scrape API 系统将持续维护和更新数据解析逻辑，从而显著降低您在电商数据集成与维护方面的成本。通过简单的API密钥调用，即可快速获取实时、准确的电商数据。

🚀 快速开始

集成 Pangolin Scrape API 非常简单。只需几步，即可开始采集您需要的电商数据：

  1. 获取 API 密钥:
    首先，您需要注册并获取您的专属 API 密钥。请访问 授权接口 或官网指引操作。 
    # 示例：获取Token (请替换为您的邮箱和密码)
curl -X POST http://scrapeapi.pangolinfo.com/api/v1/auth \
  -H 'Content-Type: application/json' \
  -d '{"email": "your-email@example.com", "password": "your-password"}'
  2. 调用 API 接口:
    使用获取到的 Token，即可调用数据采集接口。我们提供了详细的 API 文档和多语言 SDK 示例 (Python, Node.js) 帮助您快速上手。 
    # 示例：Amazon 商品详情抓取 (请替换 YOUR_TOKEN 和商品链接)
curl -X POST http://scrapeapi.pangolinfo.com/api/v1/scrape \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -d '{
    "url": "https://www.amazon.com/dp/B0DYTF8L2W",
    "parserName": "amzProductDetail",
    "formats": ["json"],
    "bizContext": { "zipcode": "10041" }
  }'

💡 开源 Amazon 关键词页面解析器

我们乐于与开发者社区分享，因此开源了针对 Amazon 关键词搜索结果页面 的部分解析代码。这份开源代码 (Go语言实现) 旨在帮助开发者理解如何从本地已有的 Amazon 搜索结果 HTML 源文件中提取结构化的产品数据。

请注意：

  • 此开源解析器专注于解析本地 HTML 文件，它本身不包含网络请求、IP代理管理或验证码自动处理等复杂的数据采集功能。
  • 该解析器基于特定时期的 Amazon 页面结构编写。尽管我们会尽力维护，但电商网站页面结构可能随时发生变化，届时您可能需要根据实际情况自行调整代码。
  • 具体用法，请参考仓库中的 parser_test.go 文件。
